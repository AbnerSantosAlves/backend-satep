<h1 align="center">⚙️ SATEP Backend</h1>

<p align="center">
API responsável pela lógica e gerenciamento de dados do sistema SATEP.
</p>

<p align="center">
O backend fornece os serviços necessários para o funcionamento do aplicativo,
processando requisições do frontend e gerenciando os dados do sistema.
</p>

<hr>

<h2>📋 Sobre o Projeto</h2>

<p>
O <strong>SATEP</strong> é um sistema desenvolvido para otimizar o agendamento
de transporte de pacientes para consultas médicas.
</p>

<p>
Este repositório contém o <strong>backend da aplicação</strong>, responsável
por gerenciar os dados, regras de negócio e comunicação com o frontend.
</p>

<hr>

<h2>⚙️ Funcionalidades da API</h2>

<ul>
<li>Cadastro e gerenciamento de usuários</li>
<li>Gerenciamento de agendamentos</li>
<li>Controle de solicitações de transporte</li>
<li>Integração com o frontend da aplicação</li>
</ul>

<hr>

<h2>🛠 Tecnologias Utilizadas</h2>

<ul>
<li>Python</li>
<li>SQLAlchemy</li>
</ul>

<hr>

<h2>📂 Estrutura do Projeto</h2>

<pre>
backend-satep/
│
├── src/
│   ├── controllers/   # Controladores da API
│   ├── services/      # Regras de negócio

│

└── README.md
</pre>

<hr>

<h2>🚀 Como executar o projeto</h2>

<h3>1️⃣ Clonar o repositório</h3>

<pre>
git clone https://github.com/AbnerSantosAlves/backend-satep.git
</pre>

<h3>2️⃣ Acessar a pasta do projeto</h3>

<pre>
cd backend-satep
</pre>

<h3>3️⃣ Instalar dependências</h3>

<pre>
npm install
</pre>

<h3>4️⃣ Executar o servidor</h3>

<pre>
npm run dev
</pre>

<hr>

<h2>🔗 Integração</h2>

<p>
Este backend é utilizado pelo projeto frontend do SATEP,
responsável pela interface do usuário e interação com o sistema.
</p>
